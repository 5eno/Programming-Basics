なお、授業では触れない細かい点ですが、中かっこ{ }の中の字下げ・インデントは、空白スペース2つ分か4つ分とするのが一般的です。次回以降は「Tabキー」あるいはスペース2ないし4つ分とするよう心がけてみてください。

・「<<」の前と後ろにそれぞれ空白を入れ、単語や記号の区切りがわかりやすいようにするとよいです。

・「main()」の後には空行を挟まずに中かっこ { を記載するのがよいでしょう。いずれ講義しますが、main() { ... } でひとまとまりなので、空白を空けずに視覚的に近接させるのがよいです。

・最後の中かっこ } は左寄せするのがよいです＝中かっこの字下げ位置を { と } で揃えるほうが一般的です。

・中かっこ閉じる } 前に空行がありますが、視覚的に冗長なスペースに見えるので、削除する方がよいでしょう。
